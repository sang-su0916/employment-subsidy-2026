# 2026 고용지원금 최적화 시스템

기업에 가장 유리한 고용지원금을 자동으로 찾아주는 웹 애플리케이션입니다.

## 🎯 주요 기능

- **8개 주요 2026 고용지원금 프로그램 지원**
  - 청년일자리도약장려금 (2026년 비수도권 청년 직접지원 신설)
  - 고령자계속고용장려금 (2026년 비수도권 우대)
  - 중증장애인고용개선장려금 (2026년 신설)
  - 정규직 전환 지원금 (2026년 재개)
  - 일손부족일자리 동행인센티브 (2026년 신설)
  - 장애인취업성공패키지
  - 국민취업지원제도
  - 육아기 10시 출근제 (2026년 신설)

- **자동 최적화**
  - 중복지급 불가 규칙 자동 처리
  - 최대 혜택 조합 자동 추천
  - 상세한 비교 분석

- **사용자 친화적 인터페이스**
  - 한국어 완벽 지원
  - 반응형 디자인 (모바일/데스크톱)
  - 인쇄 가능한 결과 보고서

## 🚀 사용 방법

### 방법 1: 브라우저에서 직접 열기

1. `index.html` 파일을 더블클릭하여 브라우저에서 엽니다.
2. 기업 정보를 입력합니다.
3. "🔍 최적 지원금 찾기" 버튼을 클릭합니다.

### 방법 2: 로컬 서버 실행

```bash
# Python 3가 설치되어 있는 경우
python3 -m http.server 8000

# Node.js가 설치되어 있는 경우
npx http-server -p 8000
```

브라우저에서 `http://localhost:8000` 접속

## 📋 필요한 정보

다음 정보를 고용보험 및 4대보험 서류에서 확인하여 입력하세요:

- 사업자등록번호
- 기업명
- 업종
- 기업 규모 (근로자 수 기준)
- 전체 근로자 수
- 고용보험 피보험자 수
- 청년근로자 수 (15-34세)
- 고령자 수 (60세 이상)
- 장애인 근로자 수
- 평균 월급여
- 최근 3개월 신규 채용 인원
- 평균 고용 유지 기간

## 💡 사용 예시

### 예시 1: 청년 채용 중소기업 (수도권)

```
사업자등록번호: 123-45-67890
기업명: 주식회사 테크스타트
업종: 정보통신업
기업 규모: 5인 이상 30인 미만
전체 근로자 수: 15명
청년근로자 수: 10명 (취업애로청년)
평균 월급여: 3,500,000원
```

**결과**: 청년일자리도약장려금 - 최대 72,000,000원 (12개월)
- 계산: 월 60만원 × 10명 × 12개월 = 7,200만원

### 예시 1-2: 청년 채용 중소기업 (비수도권)

```
사업자등록번호: 123-45-67890
기업명: 주식회사 테크스타트 대전지사
업종: 정보통신업
지역: 대전광역시 (비수도권)
기업 규모: 5인 이상 30인 미만
전체 근로자 수: 15명
청년근로자 수: 10명 (취업애로청년)
```

**결과**: 청년일자리도약장려금 - 최대 144,000,000원
- 기업 지원: 월 60만원 × 10명 × 12개월 = 7,200만원
- 청년 직접지원: 최대 720만원 × 10명 = 7,200만원 (2년간)
- **총액: 1억 4,400만원**

### 예시 2: 고령자 계속고용 기업

```
사업자등록번호: 123-45-67890
기업명: 주식회사 시니어케어
업종: 사업서비스업
기업 규모: 30인 이상
전체 근로자 수: 50명
정년 도달 후 계속고용 근로자: 5명
지역: 비수도권
```

**결과**: 고령자계속고용장려금 - 최대 72,000,000원 (36개월)
- 계산: 월 40만원 × 5명 × 36개월 = 7,200만원 (비수도권 우대, 최대 3년)

## 📊 지원금 계산 로직 (2026년 기준)

각 지원금은 다음 요소를 기반으로 계산됩니다:

- **청년일자리도약장려금**:
  - 기업: 월 60만원 × 청년 수 × 12개월 = 최대 720만원/인
  - 청년 직접지원 (비수도권): 최대 720만원/인 (2년간, 2026년 신설)
- **고령자계속고용장려금**: 월 30만원(수도권) 또는 40만원(비수도권) × 계속고용 인원 × 36개월 (최대 3년)
- **중증장애인고용개선장려금**: 월 35만원(남성) 또는 45만원(여성) × 중증장애인 수 × 12개월 (50~100인 미만, 2026년 4월 신청 시작)
- **정규직 전환 지원금**: 전환보너스 80만원 + 월 60만원 × 12개월 = 최대 800만원/인 (2026년 재개)
- **일손부족일자리 동행인센티브**: 월 50만원 × 중장년(50세+) 채용 인원 × 12개월 (2026년 신설)
- **장애인취업성공패키지**: 월 60만원 × 6개월 = 360만원/인 (개인 대상)
- **국민취업지원제도**: I유형 월 60만원 × 6개월 = 360만원/인 (개인 대상)
- **육아기 10시 출근제**: 월 30만원 × 12개월 = 360만원/인 (2026년 신설)
