# 2026 고용지원금 분석 시스템

기업 정보를 입력하면 받을 수 있는 고용지원금을 자동으로 분석해주는 웹 애플리케이션입니다.

🔗 **바로가기**: [https://subsidy-2026.vercel.app](https://subsidy-2026.vercel.app)

## 주요 기능

- **맞춤형 지원금 분석**: 기업 규모, 지역, 업종, 근로자 현황에 따른 수급 가능 지원금 자동 계산
- **중복 수급 방지**: 동일 근로자에 대한 중복 지원금 자동 제외 및 최적 조합 추천
- **실시간 계산**: 입력 즉시 예상 수급액 및 상세 산출 내역 확인
- **결과 내보내기**: Excel/PDF 형식으로 분석 결과 다운로드

## 사용 방법

1. **기업 정보 입력**
   - 지역 (수도권/비수도권)
   - 업종
   - 총 직원 수

2. **근로자 현황 입력**
   - 청년 (15~34세)
   - 중장년 (35~54세)
   - 고령자 (55세 이상)
   - 장애인 (경증/중증)
   - 육아기 근로자
   - 비정규직

3. **분석 결과 확인**
   - 수급 가능 지원금 목록
   - 예상 수급액 및 산출 내역
   - 검증 상태 (2026년 확인/미확인)

4. **결과 다운로드**
   - Excel 또는 PDF로 내보내기

## 기술 스택

- **Frontend**: HTML, CSS, JavaScript (Vanilla)
- **Charts**: Chart.js
- **Export**: xlsx, jspdf, html2canvas
- **Hosting**: Vercel (Static)
